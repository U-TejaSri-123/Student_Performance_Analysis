# Student Performance Analysis

## Objective
Analyze student performance using Python and Data Analysis techniques.

## Dataset
The dataset contains 1,000,000 student records with:
- Student ID
- Weekly Self Study Hours
- Attendance Percentage
- Class Participation
- Total Score
- Grade

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- GitHub

## Analysis Performed
- Grade Distribution
- Study Hours vs Total Score
- Attendance vs Total Score
- Class Participation vs Total Score
- Correlation Analysis

## Key Findings
- Average Score: 84.28
- Grade A students: 54.9%
- Weekly Self Study Hours have a strong positive correlation (0.812) with Total Score.
- Attendance and Class Participation showed very weak correlation in this dataset.
